Тебе будет предоставлен кусок текста из документации компании ОАО "РЖД". Твоя задача - для этого текста сделать следующее:
- составить список вопросов, на которые можно найти ответы в данном тексте. Вопросов ДОЛЖНО быть как можно БОЛЬШЕ, но при этом вопросы должны быть разнообразные и не повторяться по смыслу. Старайся придумать примерно по 2-4 вопроса на каждое предложение. Вопросы не должны быть привязаны к какому-либо контексту, примеры плохого вопроса: "Кто является работниками в контексте договора?", "Какие понятия используются в настоящем Договоре?". Пример хорошего вопроса: "Что такое коллективный договор ОАО «РЖД»?".
- составить список ключевых слов для данного текста. Ключевое слово может состоять из нескольких слов (обычно от 1 до 3 слов). Список должен быть как можно большим, но при этом ключевые слова должны быть разнообразные и не повторялись по смыслу.
- составить краткое содержание для данного текста в одно предложение. Краткое содержание должно передавать смысла текста. Старайся сокращать изначальный текст и использовать слова из самого текста.

Контекст:
"""
{context}
"""

Текст:
"""
{chunk}
"""


Ответ дай в формате JSON с полями:
"questions" - список вопросов на русском языке;
"keywords" - список ключевых слов на русском языке;
"summary" - краткое изложение текста на русском языке.

Ответ дай в формате JSON без объяснений.
