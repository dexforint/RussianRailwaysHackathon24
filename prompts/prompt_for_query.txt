Тебе будет предоставлен вопрос (запрос) пользователя (сотрудника компании) по документации компании ОАО "РЖД". Твоя задача - для этого вопроса сделать следующее:
- составить список парафраз для данного запроса. Данные парафразы должны быть таковы, что могут заменить оригинальный вопрос пользователя без потери смысла. 
- составить список ключевых слов для данного текста. Ключевое слово может состоять из нескольких слов (обычно от 1 до 3 слов). Список должен быть как можно большим, но при этом ключевые слова должны быть разнообразные и не повторялись по смыслу.
- составить предположительный (гипотетический) ответ для данного вопроса пользователя.

Контекст:
{context}

Пример запроса:
"Размер оплаты за посещение дошкольного учреждения."

Примеры парафраз:
"Сколько мне будет необходимо платить за корпоративный детский сад?"
"сколько платить за ребенка в корпоративном детском саду"
"Оплата за детский сад"
"сколько платить за ребенка в детском саду"
"Стоимость оплаты за корпоративный детский сад со стороны работника"


Ответ дай в формате JSON с полями:
"paraphrases" - список текстов, перефразированных по запросу (query);
"keywords" - список ключевых слов;
"hypothetical_answer" - краткое изложение текста.

Ответ дай в формате JSON без объяснений (твой ответ должно начинаться на символ "{" заканчиваться символом "}").
